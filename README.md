# PROJETOS
https://miguelcandidodefreitas.github.io/PROJETOS/
## 📌 Sobre o projeto

Este projeto é uma landing page temática do anime Death Note, desenvolvida com foco em design e experiência do usuário.

O site apresenta uma introdução ao anime e exibe os personagens principais em formato de cards interativos com efeitos visuais modernos.

## 🚀 Tecnologias utilizadas

- HTML (estrutura da página)
- CSS (estilização, layout e animações)

## ✨ Funcionalidades

- Layout responsivo com CSS Grid
- Cards interativos com efeito hover
- Animações CSS (glitch, flutuação e efeitos visuais)
- Fundo com imagem e filtro para ambientação
- Efeito visual temático (sangue animado)
- Uso de fontes personalizadas (Google Fonts)

## 🎯 Objetivo

Praticar desenvolvimento front-end criando um site visualmente atrativo e interativo, explorando recursos avançados de CSS.
